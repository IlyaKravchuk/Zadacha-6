2.2 Объектно-ориентированное программирование.

Условие задачи:

1) Определить класс с двумя/тремя закрытыми полями. 2) Организовать доступ к закрытым членам класса через свойства. 3) Определить и заполнить список экземпляров класса (не менее 5 элементов). 4) Отсортировать список по одному из параметров (выбрать самостоятельно). 5) Добавить новый элемент в конец списка, добавить новый элемент в произвольную позицию списка (выбрать самостоятельно). 6) Удалить оба ранее вставленных элемента. 7) Найти элемент списка по определенному значению одного из его членов (значение задать самостоятельно). 8) Очистить список.

Программа работает на языке С#. В программе определяется класс с двумя закрытыми полями,организуется доступ к закрытым членам класса,заполняется список класса,сортируется по параметру "Длина грани",либо по параметру "Длина стороны основания",добавляются два новых элемента (один в конец списка,другой на задаваемую пользователем позицию),затем добавленные элементы удаляются,находится элемент списка по параметру "название",а после список очищается.

Входными данными программы являются:

Натуральные целые числа, определяющие критерии сортировки,номер позиции,поиска и значения критерия поиска
Примечание:

Число, определяющее номер позиции должно быть целочисленным.
Только соблюдение всех условий, оговоренных в примечании, гарантирует безошибочную и стабильную работу программы.
